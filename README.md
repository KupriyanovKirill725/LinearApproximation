# LinearApproximation
Программа для линейной аппроксимации данных методом наименьших квадратов (МНК) с ручным вводом данных!

## Описание

Программа позволяет:
- Вводить точки данных вручную через консоль
- Вычислять уравнение линейной регрессии методом МНК
- Оценивать качество модели с помощью коэффициента детерминации R²
- Вычислять среднеквадратичную ошибку (RMSE)
- Получать автоматическую оценку качества модели

## Установка и запуск

**1. Клонирование репозитория**

**2. Установка зависимостей**
    
bash

**3. Запуск программы**
     
bash
