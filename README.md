# LinearApproximation
Программа для линейной аппроксимации данных методом наименьших квадратов (МНК) с ручным вводом данных!

## Описание

Программа позволяет:
- Вводить точки данных вручную через консоль
- Вычислять уравнение линейной регрессии методом МНК
- Оценивать качество модели с помощью коэффициента детерминации R²
- Вычислять среднеквадратичную ошибку (RMSE)
- Получать автоматическую оценку качества модели

## Для чего нужна линейная аппроксимация
1. **Выявление закономерностей.** Помогает определить характер зависимости между переменными, выявить устойчивые тренды за шумом и случайными колебаниями.
2. **Прогнозирование.** Позволяет предсказывать будущие значения на основе исторических данных.
3. **Упрощение анализа.** Сложные зависимости заменяются более простыми моделями, что облегчает вычисления и интерпретацию результатов.
4. **Сжатие информации.** Сохраняет ключевые закономерности при уменьшении объёма данных.
5. **Восстановление пропущенных значений.** Может использоваться для заполнения пробелов в данных.
